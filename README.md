# Micro-CT Image Super-Resolution of Rock Samples

This repository contains the code and supporting materials for the research project:

“Application of Machine Learning for Super-Resolution of Micro-CT Images of Rock Samples”

This project focuses on applying machine learning-based super-resolution methods to enhance the spatial resolution and visual quality of micro-CT images of rock samples.

The repository is currently under active development. Source code, preprocessing workflows, evaluation scripts, and example results will be uploaded during the review and revision stages of the manuscript.

---

## Research Scope

The main objective of this project is to improve the resolution and quality of rock micro-CT images using machine learning methods.

The scope of this repository is limited to:

- Micro-CT image super-resolution
- Image enhancement
- Preprocessing of micro-CT image data
- Model training and testing workflows
- Quantitative image-quality evaluation
